UX

"De student heeft de interface van het mediaproduct getoetst aan een door hem gekozen set design guidelines en op basis daarvan het interface ontwerp verbeterd."

Ik heb een UI gemaakt volgens de Android-guideline. Om inzicht te krijgen naar hoe je een interface moet maken voor een Android device. Hier kreeg ik de leren hoe ik een design maak door een guideline te volgen. Ik kwam er zo achter hoe belangrijk spacing is. Ook heb ik termen zoals Nested Doll en Hub en Spoke geleerd die bij mobile devices terugkomen. Door de feedback terug te hebben gekregen op mijn device weet ik voor de volgende keer dat ik ook moet letten op wereldwijde icons.

Deze android alles app was om te leren hoe ik een guideline moest volgen. Omdat Jurren, die de grootste gedeelte van de design deed van UI 2, al wist hoe hij zo een UI moest maken, heb ik deze UI van de alles app gemaakt om dus te leren waarom de keuzes zijn gemaakt van font size en spacing.

Feedback dat ik kreeg hoe de alles app is geworden zoals het is. Men waar ik de UI had getest vonden de kleuren niet mooi, dus ik heb het darkmode gemaakt. Om te weten welke kleuren darkmode moesten zijn heb ik inspiratie genomen van darkmode van Reddit en wat er in de guideline stond. Als ik het nu zou moeten veranderden nog op docenten feedback. Zou ik andere, globale icoontjes gebruiken.

Ook heb ik (niet echt een interface) een mediaproduct getoetst met de doelgroep om daar uiteindelijk met de gekregen feedback een product te maken.
