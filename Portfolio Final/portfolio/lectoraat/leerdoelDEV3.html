Ik heb voor al mijn programmeer projecten, inclusief mijn portfolio, visual studio code gebruikt. Dit is omdat ik visual studio code al gewend ben van vorige semester. Ik gebruik het nog steeds omdat daar de plugin emmet standaard erin zit, maar je kunt ook vanuit deze software ook andere plugins makkelijk binnen de app downloaden.

Ik heb geen andere editor gebruikt, omdat ik vooral met html, CSS en javascript/REACTJS bezig ben geweest bij Media. Ik heb wel in semester 2 sublime tekst gebruikt, deze vond ik ook wel prima, het nadeel is dat extensions minder makkelijk te downloaden waren en het was een tekst editor. Visual studio code kan ik bijna voor alles gebruiken, dus mocht ik iets nieuws proberen (zoals REACTJS) kan ik gelijk makkelijk npmâ€™s en nieuwe extensions downloaden.

Extensions die ik gebruik zijn bijvoorbeeld voor mij om minder dingen te typen, of om dingen overzichtelijk te maken. Babel javascript gebruik ik voor ReactJS. Bracket Pair Colorizer 2 gebruik ik omdat ik van kleurtjes hou en dingen overzichtelijker maakt voor mij. Discord Presence gebruik ik ook om mijn vrienden online te laten weten aan wat voor een soort bestand ik aan het werken ben en dat ik bezig ben DUS NIET MIJ STOREN. Ook gebruik ik prettier, omdat ik dan geen moeite hoef te doen in mijn code mooi maken als het ook met 1 druk van de knop kan. Tot slot gebruik ik live server om een local server te starten en mijn applicaties te testen.
